# react-native-khalti-sdk

react-native wrapper for khalti iOS and android sdks

## Installation

```sh
npm install react-native-khalti-sdk
```

## Usage

```js
import KhaltiSdk from "react-native-khalti-sdk";

// ...

const result = await KhaltiSdk.multiply(3, 7);
```
